🎬 Spotify Canvas Checker

See which tracks in a playlist have a Canvas video

Setup instructions (required — read before using)

Why login is needed: Canvas data is not available via the public Spotify API. This app uses an unofficial endpoint that requires a real user token.

Setup (one time):

  1. Go to developer.spotify.com/dashboard and create a free app.
  2. In your app settings, add as a Redirect URI. Copy the exact path shown.
  3. Set the app type to Web API. Copy your Client ID.
  4. Paste the Client ID below and click Log in with Spotify.

Privacy: Your token is stored only in your browser's sessionStorage and never sent anywhere except Spotify's servers.

Spotify login

Paste your Client ID from the Spotify Developer Dashboard, then log in. The redirect URI to whitelist is shown in the setup instructions above.
